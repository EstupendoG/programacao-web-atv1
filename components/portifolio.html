Rolador de Dados

Esse projeto nasceu da ideia de criar uma ferramenta simples e divertida para jogadores de RPG. O Rolador de Dados é um site que permite escolher a quantidade de dados, o número de faces e até aplicar modificadores nos resultados.

Além disso, ele conta com um histórico de rolagens, exibição do maior e menor valor obtido e uma personalização visual com diferentes temas e paletas de cores. Tudo pensado para tornar a experiência prática e visualmente agradável.

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, servindo como um ótimo exercício para treinar manipulação de DOM, lógica de programação e criação de interfaces interativas.

Projeto Schiavon

O Projeto Schiavon foi um trabalho fanmade criado durante o curso de Análise e Desenvolvimento de Sistemas na Fatec São José dos Campos. O objetivo era desenvolver um site voltado para apresentar a carreira e as produções do músico Yun Li.

A proposta incluía reunir informações sobre a trajetória do artista, suas músicas e discografia, tudo organizado em uma interface simples, clara e responsiva. O foco principal era aplicar boas práticas de front-end e entregar um material visualmente atraente.

Esse projeto serviu como aprendizado em HTML, CSS e organização de conteúdo, além de reforçar a importância da experiência do usuário na hora de estruturar páginas informativas.
